Rest Counter (C#, Windows Forms Application)

My rest counter app counts music for the user, allowing them to take break without counting long rests!

the user can record their own music onto the application by entering the bpm, beats per bar, and the number of bars of rest or music.

the user can then save the piece or click play to begin!

when the user clicks 'save', three arrays are saved onto an xml file by loop to be accessed later.

the player can be paused by pressing space.
